Hardware

Hardware é o termo utilizado para se referir a toda a parte física de um sistema de computação. É o conjunto de componentes eletrônicos, circuitos integrados e placas, que se comunicam através de um sistema de instruções. O hardware é basicamente tudo o que você pode tocar em um computador. A ciência que estuda o hardware é conhecida como arquitetura de computadores.

Em linhas gerais, as diferentes partes do hardware são:

os periféricos de entrada - teclado, mouse, monitor touch, ...
o sistema central - CPU, memórias principal e secundária
os periféricos de saída - discos, impressora, monitor, ...

Os principais componentes de hardware incluem:

Unidade Central de Processamento (CPU): É o cérebro do computador, onde a maior parte dos cálculos acontece.
Memória: Armazena informações que o computador usa.
Periféricos de Entrada/Saída: Permitem ao usuário interagir com o computador.
