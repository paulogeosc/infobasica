Memória RAM e ROM

ROM

A Memória Somente de Leitura (ROM) é um tipo de memória que contém instruções de inicialização para o computador - verificar se todo o sistema está em condições de uso, quais os periféricos estão conectados ao sistema e carregar o sistema operacional.

Ela só pode ser lida e não é volátil, ou seja, as informações permanecem armazenadas mesmo quando o computador é desligado.

RAM

A Memória de Acesso Aleatório (RAM) é um tipo de memória que permite a leitura e a escrita de dados. É a memória principal do computador, onde os dados são armazenados temporariamente enquanto o computador está ligado.

Quando o computador é desligado, todos os dados na RAM são perdidos.

A quantidade de RAM disponível em um sistema afeta diretamente a velocidade de processamento. Mais RAM permite que o sistema operacional e os programas armazenem mais dados temporariamente, reduzindo a necessidade de acessar o disco rígido. Isso resulta em uma resposta mais rápida ao executar vários programas simultaneamente ou ao lidar com tarefas intensivas em uso de memória. No entanto, uma vez que uma quantidade suficiente de RAM está disponível, o aumento adicional pode ter benefícios marginais ou insignificantes em termos de velocidade de processamento. Outros componentes do sistema, como o processador, também influenciam o desempenho geral.

De acordo com os requisitos mínimos de sistema anunciados pela Microsoft em junho de 2021, o mínimo de RAM necessário para rodar o Windows 11 é de 4 gigabytes (GB).
