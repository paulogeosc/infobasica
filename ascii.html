Tabela ASCII

A Tabela ASCII (American Standard Code for Information Interchange, ou Código Padrão Americano para o Intercâmbio de Informação) é uma tabela de caracteres que usa números de 0 a 127 para representar letras, números, pontuação e outros caracteres especiais. Cada caractere é representado por um número de 7 bits, embora geralmente sejam usados 8 bits (um byte) para facilitar o processamento no computador.
