Componentes do Computador

Conceitos

CPU

A CPU (Central Processing Unit ou Unidade Central de Processamento) é o cérebro do computador. Ela executa as instruções dos programas, realizando operações matemáticas e lógicas.

Memória ROM

A memória ROM (Read-Only Memory ou Memória Somente de Leitura) armazena instruções que são lidas mas não podem ser alteradas. Ela contém o firmware, que é o software básico que o computador precisa para iniciar.

Memória RAM

A memória RAM (Random Access Memory ou Memória de Acesso Aleatório) é a memória principal do computador. Ela armazena temporariamente os dados que a CPU precisa para executar as operações. A RAM é volátil, o que significa que os dados são perdidos quando o computador é desligado.

Unidades de entrada

As unidades de entrada permitem que os usuários insiram dados no computador. Exemplos incluem teclado, mouse, scanner e microfone.

Unidades de saída

As unidades de saída permitem que os usuários recebam informações do computador. Exemplos incluem monitor, impressora, alto-falantes e fones de ouvido.
