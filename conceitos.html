Hardware

Hardware é o termo utilizado para se referir a toda a parte física de um sistema de computação. É o conjunto de componentes eletrônicos, circuitos integrados e placas, que se comunicam através de um sistema de instruções. O hardware é basicamente tudo o que você pode tocar em um computador.
De maneira geral, podemos classificar os componentes principais de hardware como sendo as unidades de entrada, sistema central, e unidades de saída.

Software

Software é o conjunto de instruções que fazem o hardware funcionar. Sem software, o hardware de um computador seria inútil. Os principais tipos de software incluem:

  • Software de Base: instalado na ROM do computador, é responsável pela checagem do hardware e carga do sistema operacional (boot)
  • Utilitários: programas que oferecem recursos específicos para melhorar o desempenho, a segurança e a eficiência do sistema.
  • Aplicativos: programas de computador desenvolvidos para realizar tarefas específicas ou fornecer funcionalidades específicas aos usuários.

Sistema Operacional

software essencial que atua como uma camada intermediária entre o hardware de um computador e os aplicativos que são executados nele. Ele gerencia recursos, como processadores, memória e armazenamento, fornece uma interface para interação com o usuário e permite a execução de aplicativos. O sistema operacional também gerencia arquivos, processos e oferece recursos de segurança para proteger o computador e os dados. É uma parte fundamental de qualquer dispositivo computacional.

Medidas de Armazenamento

Assim como os sistemas de medidas e de pesos, que utilizam o grama (g), o quilograma (Kg), o centímetro (cm) e o metro (m), a Tecnologia de Informação utiliza suas próprias unidades de medidas: os dados (informações) do computador são medidos em bytes (B) e seus correspondentes (KB, MB, GB, TB).
